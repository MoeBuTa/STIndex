STIndex

Spatiotemporal Information Extraction System

Problem Statement

Challenge: Unstructured text contains valuable spatiotemporal information that is difficult to extract and normalize systematically.

  • Temporal expressions vary widely: "March 15, 2022", "last Tuesday", "three weeks ago"
  • Spatial entities need disambiguation: "Paris" (France, Texas, or Ontario?)
  • Traditional rule-based systems struggle with context and ambiguity
  • Separate extraction pipelines for temporal and spatial data are inefficient

Project Objectives

Goal: Build an intelligent system that extracts and normalizes both temporal and spatial information from unstructured text using Large Language Models (LLMs).

  • Single unified extraction call for both temporal and spatial entities
  • Normalize temporal expressions to ISO 8601 format
  • Geocode spatial entities with context-aware disambiguation
  • Support multiple LLM providers (OpenAI, Anthropic, HuggingFace)
  • Provide structured, validated outputs for downstream applications

Next Steps

Dataset Construction

  • Expand evaluation dataset from 100 to 1000+ documents with diverse text sources
  • Manual annotation by domain experts with ground-truth ISO 8601 normalization and geocoding
  • Synthetic data augmentation using GPT-4/Claude for challenging cases (ambiguous references, temporal coreference)
  • Balance across temporal types (DATE, TIME, DURATION, SET) and spatial types (CITY, STATE, COUNTRY)

Model Fine-Tuning

  • Qwen3-8B (Primary): LoRA/QLoRA fine-tuning to improve temporal recall (80%→90%+) while maintaining perfect spatial extraction
  • Qwen3-4B-Thinking (Secondary): Chain-of-thought fine-tuning to improve geocoding accuracy (64.5%→85%@25km) through better parent region extraction
  • Multi-GPU distributed training with DeepSpeed ZeRO-2

Evaluation & Release

  • Benchmark on standard datasets (TempEval-3, ACE TERN, GeoCorpora) and cross-domain evaluation
  • Target: 95%+ combined F1, 90%+ temporal recall, 85%+ geocoding accuracy@25km
  • Open-source release of fine-tuned models and training datasets
